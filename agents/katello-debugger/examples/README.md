# Example Debugging Scenarios

This directory contains example debugging scenarios that the Katello Debugger agent can help with.

## Scenario Examples

### 1. Repository Sync Failure (`sync-failure-example.md`)
A repository synchronization that fails with SSL certificate errors.

### 2. Content View Publishing Issue (`content-view-publish-example.md`)
A content view that fails to publish due to package dependency conflicts.

### 3. Host Registration Problem (`host-registration-example.md`)
A host that cannot register due to activation key configuration issues.

### 4. Performance Investigation (`performance-debug-example.md`)
Investigating slow API responses and database query performance.

## How to Use These Examples

Each example file contains:
- **Problem Description**: What the user is experiencing
- **Error Messages**: Relevant logs and error output
- **Agent Response**: How the Katello Debugger agent would analyze and solve the issue
- **Solution**: Step-by-step resolution process
- **Prevention**: How to avoid similar issues in the future

These examples serve as templates for common debugging patterns and demonstrate the agent's capabilities.
