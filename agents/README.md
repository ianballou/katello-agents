# Claude Agents for Katello Development

This directory contains Claude agents designed to assist with Katello development and debugging.

## Available Agents

### katello-debugger
A specialized agent for debugging and fixing Katello issues. This agent can help with:
- Analyzing Katello codebase issues
- Identifying common bug patterns
- Suggesting fixes for Katello-specific problems
- Understanding Katello architecture and dependencies

## Usage

Each agent directory contains its own configuration and instructions for use with Claude.
